# База доменів

Невеликий навчальний застосунок для обліку доменів. Серверна частина написана на `Node.js + Express`, клієнтська — на `React`. Застосунок отримує список доменів з API, дозволяє додати новий запис через форму та одразу оновлює таблицю на сторінці.

У сервері реалізовано маршрути `GET /api/domains` і `POST /api/domains`, валідацію полів та `CORS` для локального фронтенда. На клієнті використано `useState`, `useEffect`, `fetch` і контрольовані інпути.

## Запуск

1. Встановити залежності:
   `npm run install:all`
2. Запустити сервер:
   `npm run dev:server`
3. Запустити клієнт:
   `npm run dev:client`

Після запуску:
- фронтенд: `http://localhost:5173`
- бекенд: `http://localhost:4000`

## Перевірка

Перевірено отримання списку доменів, додавання нового запису та спрацювання валідації при некоректних даних.
